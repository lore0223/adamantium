Las barras de navegación pueden utilizar .navbar-toggler, .navbar-collapsey las .navbar-expand{-sm|-md|-lg|-xl}clases para cambiar cuando su contenido se colapsa detrás de un botón. En combinación con otras utilidades, puede elegir fácilmente cuándo mostrar u ocultar elementos particulares. Para las barras de navegación que nunca se colapsan, agregue la .navbar-expandclase en la barra de navegación. Para las barras de navegación que siempre colapsan, no agregue ninguna .navbar-expandclase.
